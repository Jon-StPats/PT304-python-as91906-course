# Python Programming (VS Code + GitHub Classroom)

This repository is a **learning course** to build Python skills for **AS91906** (Level 3 Digital Technologies).

## How to use this repo
- Work through folders **in order**.
- Read the comments, then complete the **TODO** exercises.
- **Commit often** (small commits are best).
- Keep your code readable: good names, clear comments.
- Mark the checkpoints as you go.

## What you need
- VS Code
- Python 3.10+ installed

## Assessment note
This repo is **not** the AS91906 assessment. It is preparation.
